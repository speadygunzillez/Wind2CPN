# Wind2CPN
Converts PC Wind Plot Tracks over to GPX for OpenCPN software to use via python

Usage: ./wind2cpn.py
Input and Output folders are hardcoded will have to change made it pretty obvious where they are may possibly updat e to be more end user friendly in the future


To Get .txt files
open PC Wind Plot
Go to Tracks > Search and Edit > Export > Export as Text File with header and data
